# Digital Audit & Assurance — Big 4 Deliverables & Workpapers

An illustrative, end-to-end engagement file for a **Digital Audit & Assurance** profile
(Chartered Accountant / Big 4 perspective). All data is fictitious and for
education/demonstration only.

## What's inside

- **`Digital_Audit_Assurance_Deliverables_Big4.xlsx`** — a 17-sheet Excel workbook of
  sample workpapers and deliverables.
- **`build_audit_workbook.py`** — the generator script (reproducible; requires `openpyxl`).

## Workbook contents

| Ref | Sheet | Purpose |
|-----|-------|---------|
| WP-01 | 01_Domain_Overview | Sub-domains, roles & skills |
| WP-02 | 02_Engagement_Lifecycle | End-to-end audit phases mapped to Standards on Auditing |
| WP-03 | 03_Acceptance_Independence | Client acceptance & independence gate |
| WP-04 | 04_Risk_&_Materiality | Risk matrix + materiality computation |
| WP-05 | 05_ITGC_Testing | IT General Controls testing |
| WP-06 | 06_Application_Controls | Automated / application control testing |
| WP-07 | 07_JE_Data_Analytics | Journal-entry testing via data analytics (CAAT) |
| WP-08 | 08_Substantive_Sampling | Substantive test of details (revenue / AR) |
| WP-09 | 09_Deliverables_Tracker | Master deliverables tracker |
| WP-10 | 10_Findings_Log | Control deficiencies & findings register |
| WP-11 | 11_KPIs_&_Insights | Engagement KPIs + partner insights |
| WP-12 | 12_Tools_&_Tech | Digital audit tool stack |
| WP-13 | 13_ESG_Assurance | Limited assurance over ESG metrics (ISAE 3000) |
| WP-14 | 14_SOC1_RCM | SOC 1 (ISAE 3402) risk & control matrix |
| WP-15 | 15_Cyber_Maturity | Cyber maturity assessment (NIST CSF 2.0) |
| WP-16 | 16_Audit_Report | Illustrative independent auditor's report |

## Regenerate

```bash
pip install openpyxl
python3 build_audit_workbook.py
```
